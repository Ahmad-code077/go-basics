# Go Basics

Welcome to my Go learning repository!

This repository documents my journey of learning Go from the fundamentals to more advanced concepts. Each topic is organized into its own folder with examples, exercises, and small projects to reinforce the concepts.

## Topics

- Variables
- Input / Output
- Conditionals (If Else, Switch)
- Loops
- Arrays
- Strings
- Slices
- Maps
- Functions
- Pointers
- Structs
- Interfaces
- Packages
- Error Handling
- Concurrency (Goroutines, Channels)
- File I/O

The goal is to build a solid understanding of Go before diving deeper into Data Structures and Algorithms.
